# GAMSS

### 하루의 감정을 기록하고 AI가 분석해주는 감정 다이어리

온디바이스 AI 기반 감정 분석과 요약을 통해
나의 하루를 더욱 쉽게 돌아볼 수 있습니다.

</div>

## 🛠️ Tech Stack

| Category               | Tech                                        |
| ---------------------- | ------------------------------------------- |
| **UI Framework**       | SwiftUI, UIKit                              |
| **Architecture**       | MVVM, Clean Architecture                    |
| **Concurrency**        | Swift Concurrency (async/await)             |
| **Dependency Manager** | Swift Package Manager (SPM)                 |
| **Networking**         | URLSession                                  |
| **BaaS**               | Firebase                                    |
| **Authentication**     | Firebase Authentication, Sign in with Apple |
| **Crash Reporting**    | Firebase Crashlytics                        |
| **On-Device AI**       | Core ML (Model TBD)                         |

## 📏 Conventions

- 코딩 컨벤션: [StyleShare Swift 컨벤션](https://github.com/StyleShare/swift-style-guide)
